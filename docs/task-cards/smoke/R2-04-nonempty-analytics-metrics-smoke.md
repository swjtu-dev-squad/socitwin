## 1. 背景

#2 的本质是核查 Analytics 页面指标是否真实工作正常；#22 要实现信息传播、群体极化、羊群效应三幅图；#24 要实现三大指标计算与分析报告。第一轮已证明页面未实现项基本诚实占位，且三大指标在 smoke 版可计算。但第一轮的一个限制是行为产出偏弱。第二轮要在非空数据场景下重新验证 Analytics，否则图表再漂亮也可能是在分析空气。

## 2. 目标

在 R2-01 已确认非空产出的前提下，验证：

*   极化曲线在真实行为下可变化；
*   传播速度不再长期接近 0；
*   羊群指数在不同配置下有区分度；
*   页面没有伪数据冒充真数据；
*   页面展示能追溯到 `status`/`history`/`agents`/`logs`。

## 3. 范围

| In Scope | Out of Scope |
|---|---|
| Analytics 页面 | 完整传播图系统 |
| Polarization Trend | SHAP / Sobol / 因果分析 |
| 传播速度 | 自定义数据集图表 |
| 羊群指数 | |
| 占位项真伪核查 | |

## 4. 前置条件

*   R2-01 通过，已获得非空行为数据。
*   前后端服务稳定运行。
*   日志重复 bug 不回归。

## 5. 执行步骤

**(1) 运行非空配置**

采用 R2-01 的配置 B，至少 10-20 步。

**(2) 进入 Analytics 页面**

检查：
*   Polarization Trend 图
*   传播速度展示
*   羊群指数
*   未实现占位项

**(3) 导出对照数据**

从 `history` / `status` / `logs` 导出同批次数据，做页面与源数据映射。

**(4) 形成真值表**

建议表头：`| 指标 | 页面值 | 数据源 | 计算方式 | 是否可信 | 备注 |`

## 6. 通过标准 (Gate)

*   [ ] 极化趋势图有真实波动
*   [ ] 传播速度在非空场景下可观察
*   [ ] 羊群指数不再恒定死值
*   [ ] 页面值可追溯到源数据
*   [ ] 未实现项仍明确标注，不伪装
*   [ ] 非空场景下页面无错误

## 7. 失败标准

*   非空场景下指标仍长期为 0 或常数。
*   页面与源数据不一致。
*   伪数据重新出现。
*   指标定义和页面口径不一致。

## 8. 证据要求

*   `artifacts/smoke/r2_04_analytics_truth_table.md`
*   `artifacts/smoke/r2_04_metrics.csv`
*   页面截图与 hover 数值截图

## 9. 备注

这张卡的重点不是“图会不会动”，而是“图动起来时是不是说真话”。
