# Round 6: 实验平台前端化

第六轮工作的目标，是将 Round 5 已经形成的“数据集 × 推荐策略 × 指标输出”的实验脚本能力，前端化为一个最小可交互实验控制台：用户能够在前端选择数据集与推荐器发起实验、比较不同实验结果的传播/极化/羊群差异，并浏览历史实验及其配置与结果，从而把当前系统从“会生成实验报告”推进为“可在产品中直接操作的实验平台”。

## 任务卡

- [R6-01: Experiment Runner 前端控制台](./R6-01-EXPERIMENT-RUNNER-CONSOLE.md)
- [R6-02: Compare Panel 前端接入](./R6-02-COMPARE-PANEL-FRONTEND.md)
- [R6-03: 实验结果归档与浏览前端化](./R6-03-EXPERIMENT-ARCHIVE-UI.md)

## 推荐执行顺序

1.  **R6-01**: 先让用户能在前端“发起实验”
2.  **R6-02**: 再让用户能在前端“比较实验”
3.  **R6-03**: 最后让用户能在前端“回看实验”
