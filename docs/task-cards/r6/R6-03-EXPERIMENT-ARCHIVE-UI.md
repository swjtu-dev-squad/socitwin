## 1. 背景

R5-03 已经做出了“可复现实验模板”，说明实验配置、结果目录、脚本都已经规范化了。
下一步自然该做的，就是让这些实验结果能在前端被查看、筛选、复盘，而不是只存在文件夹或终端里。

## 2. 目标

让用户在前端浏览历史实验结果，包括：

- 实验名称、数据集、推荐器、seed / steps
- 关键指标摘要
- 查看详情与导出结果

## 3. 范围

| In Scope | Out of Scope |
|---|---|
| 实验历史表格 | 任务暂停/恢复控制 |
| 基础筛选 | 权限系统 |
| 详情抽屉/详情页 | 多用户审计日志 |
| 配置查看 | 完整批量调度中心 |
| 关键结果摘要 | |
| JSON/Markdown 导出链接（最小版） | |

## 4. 建议最小列表字段

`experimentId`, `name`, `datasetId`, `recommenders`, `steps`, `seed`, `createdAt`, `summary`

## 5. 通过标准 (Gate)

- [ ] 实验历史列表可见
- [ ] 可查看单个实验详情
- [ ] 至少支持一种筛选
- [ ] 展示关键结果摘要
- [ ] 空状态与错误状态处理正常
