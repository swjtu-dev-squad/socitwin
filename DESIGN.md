# 深色科技风设计系统

## 1. 视觉主题与氛围

SoCITwin 的态势推演系统采用深色科技风格，融合了 MiniMax 设计语言与现代科技美学。整体视觉以深色为基底，通过青色霓虹光效、毛玻璃质感、精密仪器风格的细线框和等宽字体，营造出高科技、精准、专业的监控与推演平台氛围。

设计语言深受 Cyberpunk 和 Sci-Fi 用户界面启发，强调数据可视化、精密控制和实时监控的感觉。深色背景不仅减轻长时间使用的视觉疲劳，还让彩色数据指标和状态指示更加醒目。等宽字体的使用强化了"精密仪器"和"命令行控制"的技术感，而毛玻璃和发光效果则增添了现代科技美学。

**关键特征：**
- 深色主题基底（`#181e25`）配合青色霓虹强调（`#00f2ff`）
- 等宽字体（JetBrains Mono）作为主要字体，强化科技感
- 毛玻璃效果（`backdrop-filter: blur(20px)`）营造层次感
- 细线框（1px）和微小圆角（2px）的精密仪器风格
- 多层次阴影和发光效果，增强视觉深度
- 渐变装饰线和角标，丰富视觉细节
- 刻度尺风格的滑块和等宽数字显示

## 2. 颜色系统

### 主色调
- **背景深色** (`#181e25`): `--color-bg-primary`，主背景色
- **次级背景** (`#27272a`): `--color-bg-secondary`，卡片背景
- **三级背景** (`#3f3f46`): `--color-bg-tertiary`，输入框背景
- **输入框背景** (`rgba(0, 0, 0, 0.3)`): 下拉框、输入框深色背景

### 强调色
- **科技青** (`#00f2ff`): 主要强调色，用于标题、装饰、发光效果
- **科技粉** (`#ea5ec1`): 次要强调色，用于控制、辅助元素
- **品牌蓝** (`#3b82f6`): `--color-minimax-primary-500`，边框、链接
- **品牌蓝悬停** (`#2563eb`): `--color-minimax-primary-600`，交互状态

### 文本颜色
- **主文本** (`#fafafa`): `--color-text-primary`，主要内容
- **次级文本** (`#a1a1aa`): `--color-text-secondary`，辅助信息
- **三级文本** (`#71717a`): `--color-text-tertiary`，描述、元数据
- **弱化文本** (`#52525b`): `--color-text-muted`，禁用状态

### 边框颜色
- **默认边框** (`rgba(59, 130, 246, 0.2)`): 细线框，半透明蓝色
- **强边框** (`rgba(0, 242, 255, 0.3)`): 科技青边框，用于输入框
- **分割线** (`rgba(59, 130, 246, 0.2)`): 区域分隔
- **边框悬停** (`rgba(59, 130, 246, 0.3)`): 交互状态

### 语义颜色
- **警示红** (`#f43f5e`): `--minimax-rose`，危险、高值
- **成功绿** (`#10b981`): `--minimax-emerald`，正常、低值
- **警告黄** (`#f59e0b`): `--minimax-amber`，中等值
- **信息蓝** (`#3b82f6`): `--minimax-blue`，信息展示

### 阴影与发光
- **标准阴影** (`rgba(0, 0, 0, 0.3) 0px 4px 6px`): 基础卡片阴影
- **发光效果** (`0 0 40px rgba(59, 130, 246, 0.1)`): 卡片发光
- **青色发光** (`0 0 10px rgba(0, 242, 255, 0.5)`): 文本、按钮发光
- **粉色发光** (`0 0 10px rgba(234, 94, 193, 0.3)`): 控制元素发光
- **深度阴影** (`rgba(0, 0, 0, 0.4) 0px 12px 16px -4px`): 浮动元素

## 3. 字体系统

### 字体家族
- **主要字体**: `JetBrains Mono`，等宽字体，科技感核心
- **备选字体**: `ui-monospace, SFMono-Regular, monospace`
- **数字字体** (可选): `DIN Alternate`，用于数字显示

### 字体层级

| 角色 | 字体 | 大小 | 字重 | 字间距 | 行高 | 用途 |
|------|------|------|------|---------|------|------|
| 标题 | JetBrains Mono | 20px | 600 | 0.15em | 1.5 | 章节标题 |
| 副标题 | JetBrains Mono | 13px | 400–500 | 0.05em | 1.5 | 描述文本 |
| 标签 | JetBrains Mono | 12–13px | 500–600 | 0.1em | 1.5 | 表单标签 |
| 数值 | DIN Alternate / JetBrains Mono | 32px | 500 | 正常 | 1 | 指标数值 |
| 按钮文本 | JetBrains Mono | 13px | 500–600 | 0.05em | 正常 | 按钮文字 |
| 微型文本 | JetBrains Mono | 10–13px | 400 | 0.02em | 1.7 | 元数据、提示 |

### 字体原则
- **等宽为核心**: JetBrains Mono 贯穿整个界面，强化科技感
- **宽字间距**: 0.05em–0.15em 字间距，增强可读性和科技感
- **适度字重**: 400–600 范围，避免过度粗体
- **数字对齐**: 使用等宽字体确保数值对齐
- **大小对比**: 32px 数值与 13px 标签形成强烈对比

## 5. 布局原则

### 间距系统
- 基本单位: 4px
- 常用间距: 4px, 6px, 8px, 16px, 24px, 32px
- 组件间距: `gap-6` (24px)
- 内边距: `p-4` (16px), `p-6` (24px)

### 网格系统
- 最大容器宽度: `1024px`
- 水平内边距: `24px`
- 主布局: `grid-cols-4` (4 列网格)
- 控制面板: `col-span-3` (占 3/4 宽度)
- 指标网格: `col-span-1` (占 1/4 宽度)

### 响应式断点
| 名称 | 宽度 | 布局变化 |
|------|------|----------|
| Mobile | <768px | 单列布局 |
| Tablet | 768–1024px | 2 列布局 |
| Desktop | >1024px | 4 列网格布局 |

### 空白使用
- **外边距**: `mb-8` (32px) 主要区域间距
- **内边距**: `p-6` (24px) 卡片内边距
- **垂直间距**: `space-y-3` (12px) 卡片间垂直间距
- **水平间距**: `gap-6` (24px) 网格列间距

### 圆角系统
- 微型 (2px): 按钮、输入框
- 小型 (4px): 图标容器、装饰元素
- 标准 (8px): 卡片、指标卡片
- 中型 (12px): 中型容器
- 大型 (16px): 大型卡片、控制面板
- 超大 (20–24px): 特大容器

## 6. 深度与阴影

| 层级 | 阴影样式 | 用途 |
|------|----------|------|
| Flat (层 0) | 无阴影 | 背景、文本块 |
| Subtle (层 1) | `rgba(0, 0, 0, 0.3) 0px 2px 4px` | 基础卡片 |
| Standard (层 2) | `rgba(0, 0, 0, 0.3) 0px 4px 6px` | 标准卡片 |
| Glow (层 3) | `0 0 40px rgba(59, 130, 246, 0.1)` | 发光卡片 |
| Elevated (层 4) | `rgba(0, 0, 0, 0.4) 0px 12px 16px -4px` | 浮动元素 |

**阴影哲学**: 使用低透明度的黑色阴影 (0.3–0.4) 保持深色主题的一致性。发光效果使用品牌色 (蓝色、青色、粉色) 的低透明度变体，创造科技感的霓虹光效。

## 7. 交互状态

### 悬停状态
- 边框透明度增加: `0.2` → `0.3`
- 背景轻微加深
- 发光效果增强
- 图标缩放: `scale-110`

### 禁用状态
- 透明度: `0.5`
- 鼠标样式: `not-allowed`
- 移除所有发光效果
- 保持基本结构可见

### 加载状态
- 使用骨架屏替代内容
- 骨架屏颜色: `rgba(59, 130, 246, 0.1)`
- 固定高度避免抖动
- 脉冲动画提示加载中

### 焦点状态
- 边框颜色: `#00f2ff`
- 发光效果增强
- 移除默认轮廓 (`outline: none`)

## 8. 动画效果

### 过渡时长
- 快速: `150ms` (下拉选项)
- 标准: `200ms` (按钮、悬停)
- 缓慢: `300ms` (大型组件)

### 缓动函数
- 标准: `ease` 或 `ease-out`
- 交互: `cubic-bezier(0.4, 0, 0.6, 1)` (脉冲动画)

### 关键帧动画

**扫描动画**
```css
@keyframes scan {
  0%, 100% { opacity: 0.3; }
  50% { opacity: 1; }
}
```

**脉冲发光**
```css
@keyframes pulse-glow {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.6; }
}
```

**闪烁效果**
```css
@keyframes flicker {
  0%, 100% { opacity: 1; }
  92% { opacity: 0.8; }
  94% { opacity: 1; }
  96% { opacity: 0.9; }
  98% { opacity: 0.8; }
}
```

**状态指示脉冲**
```css
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}
```

## 9. 设计原则

### Do (应该做)
- 使用深色背景 (`#181e25`) 作为主要背景
- 应用 JetBrains Mono 等宽字体强化科技感
- 使用青色 (`#00f2ff`) 作为主要强调色
- 采用毛玻璃效果 (`backdrop-filter: blur(20px)`) 增加层次
- 使用细线框 (1px) 和小圆角 (2px) 创造精密感
- 应用发光效果增强视觉焦点
- 使用宽字间距 (0.05em–0.15em) 增强可读性
- 保持一致的间距系统 (4px 基本单位)
- 使用渐变装饰线丰富视觉细节
- 添加装饰角标增强科技感

### Don't (不应该做)
- 不要使用浅色背景 — 深色是主题核心
- 不要使用非等宽字体 — JetBrains Mono 是科技感的关键
- 不要过度使用发光效果 — 适度使用避免视觉疲劳
- 不要使用大圆角 — 2px–16px 范围保持精密感
- 不要混用多种字体 — 专注等宽字体的一致性
- 不要使用高饱和度颜色 — 保持低饱和度、高亮度的科技感
- 不要忽视加载状态 — 使用骨架屏保持体验连贯
- 不要过度使用动画 — 适度动画增强交互，避免干扰
- 不要破坏间距系统 — 保持 4px 基础单位的倍数关系

## 10. 响应式行为

### 断点策略
| 断点 | 宽度 | 关键变化 |
|------|------|----------|
| Mobile | <768px | 单列布局，堆叠卡片 |
| Tablet | 768–1024px | 2 列网格，缩小间距 |
| Desktop | >1024px | 完整布局，4 列网格 |

### 布局折叠
- 主网格: 4 列 → 2 列 → 单列
- 控制面板: 保持全宽，内部响应式
- 指标网格: 单列布局，垂直堆叠
- 间距: 24px → 16px → 12px
- 字体: 32px 数值 → 28px → 24px
